# Security Policy

This is a small solo game project with no accounts, servers, or user data. All
game state lives in the player's own browser (localStorage).

## Reporting a vulnerability

If you find a security issue (for example in the build or deploy pipeline),
please open a GitHub issue, or use GitHub's private vulnerability reporting
(Security tab -> Report a vulnerability). Please do not include exploit details
in a public issue if the problem is sensitive.

There is no formal SLA, but reports will be looked at as time allows.
