## Completed

- PR #11: input-driven e2e playthrough of the delivery loop. Drives the courier
  with real key presses in headless Chromium (reach the home town, accept a
  contract, drive to the destination, complete the delivery) and asserts the
  rewards and save state. Added a read-only `window.__courier` hook, attached
  only when the game boots with `?e2e` and never in normal play. It exposes
  `getState()` and `nextStepToward(tile)`; movement is genuine keyboard input
  through Phaser, so it walks the same path a player would.
- PR #12: route-unlock coverage. New test drives to the Greybridge signpost and
  confirms the southern ford opens: the ford flag flips, `ford-crossing-greybridge`
  is recorded in the live unlock set and the save, and the ford tile flips from
  impassable to drivable. Extended the hook with `fordUnlocked`, the `unlocks`
  list, the `signpost` location, and an `isPassableTile(x, y)` probe.
