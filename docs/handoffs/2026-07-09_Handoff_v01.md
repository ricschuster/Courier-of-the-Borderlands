## Completed (this session, in order)

- **#62 Scrollable overlays.** New `src/scenes/scrollable-panel.ts`: the journal
  and skills overlays now use an opaque, screen-height box that wraps text to its
  width (no more left-edge spill), clips it with a geometry mask, and scrolls with
  the mouse wheel (the mask tracks the camera via scrollFactor 0). Overlay fonts
  trimmed (journal/skills 12->11, board 13->12, dialogue 13->12, summary 14->13).
  The box sits right of the status column, so it no longer covers the title.
- **#63 Locked-ford hint.** Standing beside a still-locked ford toasts "The ford
  is blocked. Reach the ford-key signpost to open this shortcut," once per
  approach, from either bank.
- **#64 In-town gateway sign.** The floating "road to X" sign is suppressed on
  tiles that also hold a town (Southmill); open-road signs are unchanged.
- **#65 Wayfinding compass.** New pure `src/systems/bearing.ts` (`bearingLabel`,
  eight-point compass from tile coordinates, so it works under fog), threaded into
  the objective line: pickup gets a heading, the carry leg reads "deliver to
  Mirewatch - head SE (12 tiles)", and the cleared-region prompt points to the
  nearest gateway. Unit-tested.
- **#66 Postmaster return greeting.** Added conditional node text to the dialogue
  engine (`textVariants` on a node, first matching flag wins, base text the
  fallback; pure and tested) and gave the postmaster a distinct greeting once the
  region is reconnected, so he no longer repeats the cold open on return.
- **#67 Docs.** Session 3 results recorded in `docs/design/05_playtest_notes.md`.
- **#68 Edge gateway label clamp.** Open-road gateway signs near the map edge are
  clamped to stay inside the map bounds ("road to Salt" no longer runs off). The
  last Session 2 signage leftover.
