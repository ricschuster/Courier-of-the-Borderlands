## Completed

### CI and test hardening
- **#116 -> #118**: Arc `push` trigger now has a `paths:` filter, so docs- and
  config-only merges no longer run the full-arc playthrough (the vector behind
  the repeated docs-merge Arc failures). Nightly stays unfiltered as a heartbeat.
  Confirmed live: tests/docs merges trigger no Arc; code/harness merges do.
- **#115 -> #119**: generalized `tapKey` to the remaining e2e specs. `pressUntil`
  now holds each attempt across a frame; the `social-skill` `k` toggles moved to
  `setSkillPanel`; `gated-contract` uses `tapKey`. Kills the one-shot-press flake
  class (validated 3/3 clean single-worker at `retries: 0`).
- **#113 -> #120**: added a data-driven capability-gated shortcut invariant
  (auto-discovers mire/tidal gates in every region: base-blocked then open, and
  a strictly shorter crossing). Completes the fast soft-lock guard the browser
  Arc used to be the only check for.

### Correctness and onboarding (from the persona debate)
- **#123 -> #127**: save-version **migration ladder**. `deserialize` no longer
  wipes every save on a `SAVE_VERSION` bump; it walks `MIGRATIONS` from the
  save's version up to current. `MIGRATIONS` is empty today (all changes so far
  additive) but is the seam for the next real reshape. Behavior preserved:
  current saves round-trip, future/versionless saves reject.
- **#124/#125/#126 -> #129 (first-run hardening)**:
  - #124 boot-error recovery overlay in `main.ts` (plain DOM outside Phaser;
    reset-save and plain-reload buttons) so a create-time throw is no longer an
    unrecoverable black canvas.
  - #125 first-run intro card: the first-ever opening toast now states the
    premise and the goal. Shown once ever via an intro-seen key kept outside the
    save; returning players get the terse status line.
  - #126 silent-save-failure warning: `writeSave` returns a `SaveResult`; the
    autosave path warns once per session when storage is unavailable or full.

### Docs
- **#128**: README refreshed to match the shipped build (three regions, seven
