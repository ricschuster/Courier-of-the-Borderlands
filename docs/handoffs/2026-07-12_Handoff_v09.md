### Difficulty selector (#135 -> #140, merged)

Exposed the travel-sink difficulty (ADR 0005) to the player. The tuning knobs
were already isolated into `WagonTuning` presets (relaxed / standard /
demanding) behind a single scene field; this wired the choice to the player.

- **G** cycles the difficulty preset (relaxed -> standard -> demanding, wrapping).
- The choice persists under its own `localStorage` key
  (`courier-of-the-borderlands/difficulty`): a durable preference that survives a
  New Game and is not part of the save.
- Applies live: the new profile drives wear, repair cost, and tank size from
  here on, so current condition is clamped to what the new profile affords
  (lowering difficulty never refills; raising never shrinks below the current
  value).
- The active difficulty shows in the control hint and a toast confirms each
  change, so an accidental press is always visible (useful while standard is
  still being dialed by playtest).
- Pure helpers (`DIFFICULTIES`, `isDifficulty`, `nextDifficulty`,
  `difficultyLabel`, `loadDifficulty`/`saveDifficulty`) unit tested; a Playwright
  spec (`tests/e2e/difficulty.spec.ts`) guards the scene wiring (key binding,
  live clamp, cross-reload persistence). ADR 0005 updated.

Verified in the built game (HUD hint reads "G: difficulty (Demanding)", Wagon
clamps 25 -> 16 on switch to demanding).
