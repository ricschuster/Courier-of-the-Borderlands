## Completed

- **#169 - Salt Runners unbuyable.** The wagon shop has 7 upgrades but only number
  keys 1-6 were registered, so the 7th (Salt Runners) rendered as affordable yet
  could not be bought. Extended the key list and added a unit guard so the shop
  cannot outgrow the keys again. Owner confirmed in-game.
- **#172 (fixes #170) - toasts covering the upgrade menu.** The toast band only
  dodged the contract board; an open blocking overlay hides the board and sits
  centred, so toasts recentred over it. Now toasts hide while a blocking overlay
  is open and reappear on close.
- **#173 (closes #171) - gradual wayfinding by difficulty.** The objective line's
  delivery help now scales: Relaxed = heading + distance, Standard = heading only,
  Demanding = neither. Owner chose the true 3-step scale. **Changes the Standard
  default** (distance now hidden).
- **#175 (closes #174) - skills onboarding.** Unspent skill points now show as a
  salient gold chip in the top-right corner (off the dense wallet line) plus a
  recurring per-level-up nudge, sequenced after the one-time first-point teach.
  Fixes the playtest finding that skills were forgotten (they were useful once
  remembered - a discoverability gap, not balance).
- **#177 + #178 (closes #176) - trail connectors.** New `trail` terrain and a
  decoupling of wear from movement speed (`wearSpeedModifier`), so a trail can
  drive like a path yet wear at marsh grade. Painted trails linking the plain
  off-road settlements to the road network across all three regions, so no
  village looks stranded. The gated pockets (Reedgrave, Saltmere, Fenholt) stay
  isolated by design. Spoke map edits were BFS-reachability-validated offline
  before editing, then re-checked in-engine.
- **#110 economy tightening - CLOSED as delivered.** The mechanism shipped in
  #134 (travel sink) + #151 (geography) and the #109 playtest validated the feel;
  no concrete constant was flagged wrong, so there is nothing left to tune without
  tuning blind. Reopen only if a future arc surfaces a specific bad number.
