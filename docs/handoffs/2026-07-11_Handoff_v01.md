## Completed

### PR housekeeping

- **#97 fixed and merged.** The Reedgrave slice-2 PR was red: `secret-to-reedgrave`
  is a sixth *standing* Greybridge contract, but `gated-contract.spec.ts` seeded
  only the original five, so `regionCleared` read false and the assertion failed.
  Added the sixth id to the seed. Rule for next time: any new standing Greybridge
  contract must be added to that spec's `BASE_GREYBRIDGE_CONTRACTS` seed list.
- **#98 (2026-07-10 handoff) merged.** It was stuck because branch protection is
  `strict: true` (branch must be up to date with main) and main had advanced;
  auto-merge does not auto-update. Fixed with the `update-branch` API call, which
  re-ran checks and let the armed auto-merge fire. Same fix applies to any PR left
  open while others merge.

### Roads-gate-the-wagon slices

- **Slice 3 - skills as keys (#99).** Repurposed the Teamster skill (speed) into
  **Off-road**: keeps a +10%/rank speed bonus and, at **rank 2**, grants the
  `mire-crossing` capability. The deep-mire shortcut now opens for coins (Marsh
  Treads) OR skill points (Off-road 2), the same route either way. This is the
  piece that finally gives skill points a real use. The core (`traversal.ts`
  `grantMet`) already supported skill grants, so the change was small: add the
  grant entry, rename the skill, retarget sample ids in specs. Note: renaming the
  id drops any `teamster` rank in an old localStorage save (`sanitizeRanks` drops
  unknown ids); acceptable for the prototype.
- **Slice 4 - arc driver spends (#100).** The full-arc e2e driver and
  `scripts/autoplay.mjs` now kit out at home: press B to buy the cheapest
  affordable upgrade, and spend skill points via the panel (Off-road first, then
  the rest). The arc asserts it finished with an upgrade owned and Off-road
  ranked, so a regression that breaks buying or ranking fails the guard. Spending
  is bounded and never blocks arc progress; the skills panel is always re-closed
  so number keys still reach the contract board.
