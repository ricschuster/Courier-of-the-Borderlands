## Completed

### Bigger later maps to stop the difficulty curve inverting (#151 -> PR #165, open)

Root cause confirmed from data: Greybridge (hub) is 30x22; the two spokes were
20x11, a third the area. Wagon wear is per-tile and dominated by terrain
roughness (road/bridge 0.06/tile, forest 0.97, marsh 1.08), so the spokes' short,
road-hugging routes barely wore the wagon and the game eased off after region 1.

- **Saltreach 20x11 -> 30x20, Fenmarch 20x11 -> 32x22.** Redrawn from scratch by
  painting features onto a grid and validating offline (BFS reachability, ford +
  tidal shortcut checks, per-route wear estimate) before touching the repo.
- Settlements spread off-road across a northern forest belt / southern marsh
  (Saltreach) and a wide road-free forest corridor (Fenmarch), so deliveries
  cross rough ground.
- Each region keeps its signature; the lagoon/mere pockets are now **sealed**
  (water wall + one dry gap at the bottom) so the tidal-flat crossing shortens
  the actual home->settlement delivery, not just an incidental bank pair.
- New region-invariant guard: each spoke's longest base home route must be >= the
  hub's, so a spoke cannot silently shrink and re-invert the curve.
- Verified: tsc + eslint clean, 705 unit tests, full chromium e2e (23) +
  full-arc + `MEASURE_DIFF=standard` all green, 0 strand events.
