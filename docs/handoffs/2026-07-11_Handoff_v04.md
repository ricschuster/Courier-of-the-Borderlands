## Completed

### Arc no longer runs on docs-only merges (#118, closes #116)

Added a `paths:` filter to the Arc `push` trigger so the ~4-16m full-arc
playthrough only runs when game code or the e2e harness changed (`src/**`,
`tests/e2e/**`, the Playwright/Vite config, lockfiles, and the workflow itself).
This was the vector behind the repeated Arc failures on docs merges: the run
fired on content it could not affect and could still trip residual e2e
flakiness. The nightly `schedule` stays unfiltered as a heartbeat. Confirmed
live this session: a tests-unit-only merge (#120) triggered no Arc run, a
game/harness merge (#119) did.

### Generalized tapKey to the remaining e2e specs (#119, closes #115)

Finished the one-shot-press hardening #114 started for the Arc. The game reads
one-shot inputs with Phaser `JustDown`, so an instantaneous down+up can land
between two starved frames and be lost.
- `pressUntil` now holds each attempt via `tapKey` instead of an instantaneous
  press (a zero-gap press could be dropped on every poll iteration), hardening
  all eight `pressUntil` callers in one place.
- `social-skill`: the two `k` toggle presses moved to the purpose-built
  `setSkillPanel` (a re-press could flip a toggle back).
- `gated-contract`: raw `press('J')` -> `tapKey`.
- Validated 3/3 clean single-worker full-suite runs at `retries: 0`.

### Capability-gated shortcut reachability tests (#120, closes #113)

The soft-lock guard is now deterministic and cheap, not only the browser Arc.
`region-invariants.test.ts` already covered no-soft-lock reachability and the
ford gates; this added the missing half, a data-driven suite that auto-discovers
every capability gate (deep-mire, tidal-flats) in every region and asserts each
is base-blocked then opens with its token, and that holding the token opens a
strictly shorter crossing. Extracted `tilesWithUnlock` / `shorterCrossings` so
the ford and capability suites share one crossing-proof. Discovers
greybridge/mire-crossing, saltreach/tidal-crossing, fenmarch/tidal-crossing
today and picks up future gates automatically.
