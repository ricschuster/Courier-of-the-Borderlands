## Completed (this session, in order)

- **#50 Road encounters (M5.3, last box).** Pure `src/systems/encounter-system.ts`
  (RoadEncounter, `pickEncounter`, one-shot via resolution flags, `outcomeForFlag`,
  `validateEncounter`) + content in `src/data/encounters.ts`. Encounters fire on a
  trigger tile, open the existing modal dialogue, and a choice applies a coin/
  reputation outcome. Resolution flags are ordinary `storyFlags` (no save change).
  Wired into `map-scene.ts`; e2e hook v6->7 with `activeEncounterId`. Unit tests +
  map-placement invariants + a browser e2e. `drive.ts` now dismisses an encounter
  that opens mid-drive.
- **#51 Three side-route encounters.** Rockfall (Greybridge 5,17), sea-fog
  (Saltreach 15,0), bogged fen-guide (Fenmarch 14,4). Pure content; invariants
  auto-cover them.
- **#52 Arc-driven contracts (M5.4).** `Contract.requires?: FlagCondition` plus
  pure `isContractAvailable` / `availableContracts` / `contractsInPlay` /
  `baseContracts`. `boardContracts()` filters by effective flags; progress counts
  use in-play contracts (never "one you cannot see"); `regionCleared()` decoupled
  to base (ungated) contracts so a revealed gated contract does not re-lock the
  spoke reveals. One gated contract per region, appearing after that region's
  reveal (`greybridge_reveal` / `saltreach_method` / `fenmarch_cost`), all to
  existing settlements. e2e hook v7->8 with `regionCleared`. Unit tests + a
  save-seeded e2e.
- **#53 Pre-playtest hardening.** `tests/unit/narrative-consistency.test.ts`
  cross-checks every gating flag against what a choice can set/derive and every
  mission reference against real content. Two fixes from adversarial review: the
  encounter toast reports coins actually spent (not the overstated nominal), and
  `openDialogue` returns whether it opened so `activeEncounter` cannot dangle.
- **#54 Hidden Road journal thread.** `src/systems/story-threads.ts`
  (`hiddenRoadProgress` / `hiddenRoadJournalLines`), a derived cross-region
  through-line over the arc-gated contracts, shown in the journal Story section
  once begun (spoiler-safe). No HUD/balance/save impact.
- **#55 DialogueController refactor.** Extracted the conversation subsystem
  (settlement talk, encounters, the modal state machine, outcomes) from MapScene
  into `src/scenes/dialogue-controller.ts` behind a narrow `DialogueHost`
  interface (host object literal). `map-scene.ts` 1713 -> 1541 lines. No behavior
  change.
